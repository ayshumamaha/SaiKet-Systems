# Word Count Tool

## Overview

The Word Count Tool is a Python-based text analysis application that calculates the number of words, characters, lines, and sentences in a text document. The project demonstrates text processing and file analysis techniques.

This application is useful for writers, students, researchers, and content creators who need quick document statistics.

---

## Features

* Word counting
* Character counting
* Line counting
* Sentence counting
* Text file analysis
* Fast processing
