# Basic File Handling

## Overview

The Basic File Handling project demonstrates essential file operations in Python including file creation, reading, writing, and appending data. The application highlights the importance of persistent data storage in software development.

The project serves as an introduction to working with external files and managing information beyond program execution.

---

## Features

* Create files
* Write data to files
* Read file contents
* Append additional information
* Manage text-based records

---

## Technologies Used

### Programming Language

* Python 3.x

### Concepts Implemented

* File Handling
* Functions
* Exception Handling
* String Manipulation

## Workflow

1. Create file

5. Display updated data

## How to Run
