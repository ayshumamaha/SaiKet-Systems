# To-Do List Application

## Overview

The To-Do List Application is a Python-based task management system developed to help users organize and track their daily activities efficiently. The application allows users to create, view, update, and delete tasks through an easy-to-use command-line interface.

This project demonstrates the practical implementation of Python programming concepts such as lists, functions, loops, conditional statements, and user input handling. It serves as a productivity tool while strengthening software development and problem-solving skills.

---

## Features

* Add new tasks
* View all existing tasks
* Mark tasks as completed
* Delete tasks
* Simple command-line interface
* Dynamic task management
* Real-time task updates

1. Launch the application
2. Display task management menu
3. Select desired operation
4. Add, view, update, or delete tasks
5. Store task information
6. Display updated task list
7. Exit application

## How to Run

1. Install Python 3.x on your system.

2. Download or clone the repository.

3. Open the project folder.

4. Run the Python script:

   python todo_list.py

5. Select options from the menu to manage tasks.
