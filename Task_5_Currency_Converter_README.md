# Currency Converter

## Overview

The Currency Converter is a Python application designed to convert one currency into another using predefined exchange rates. The project demonstrates mathematical computations and financial application development.

The system allows users to perform quick and accurate currency conversions through a simple command-line interface.

---

## Features

* Currency conversion
* Exchange rate calculations
* Multiple currency support
* Instant results
* User-friendly interface

---

## Technologies Used

### Programming Language

* Python 3.x

### Concepts Implemented

* Functions
* Dictionaries
* Mathematical Operations
* User Input Handling

---

## Workflow

1. Select source currency
2. Select target currency
3. Enter amount
4. Apply exchange rate
5. Display converted value

---

## How to Run

1. Install Python 3.x
2. Run the script
3. Enter currency information
4. View conversion results
