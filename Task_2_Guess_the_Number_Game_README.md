# Guess the Number Game

## Overview

The Guess the Number Game is a Python-based interactive game where the computer generates a random number and the player attempts to guess it within a limited number of attempts. The application provides hints to guide the player and creates an engaging gaming experience.

This project demonstrates the use of randomization, loops, conditional statements, and user interaction in Python programming.

---

## Features

* Random number generation
* Interactive gameplay
* Hint system
* Multiple attempts
* Win/Loss detection
* User-friendly interface

## Workflow

1. Generate random number
2. Accept user guess
3. Compare guess with target number
4. Provide hints
5. Repeat until correct guess or attempts exhausted
6. Display result

---

## How to Run

1. Install Python 3.x
2. Run the script
3. Enter your guesses
4. Follow the hints
5. Win by guessing correctly
